# rasa-hangman
add a game of hangman to your chatbot

```console
git clone https://github.com/dadecoza/rasa-hangman.git
cd rasa-hangman
./train_nlu.sh && ./train_core.sh
docker-compose up
```

You can open the webchat on http://localhost:8080/
